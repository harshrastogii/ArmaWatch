ARMAWATCH — SIMPLE VERSION (map only)
=====================================

This is the upgraded weapons map, ready to go on your existing web page.
It replaces the old map but keeps the same page and web address.

WHAT PEOPLE CAN DO WITH IT
- Search for a company by name
- Filter by type of site or program
- Click any dot to see the company's details and sources
- Switch between a normal map and a satellite (aerial) view
- See how many weapons sites are near them
- Download the list, or share a company on social media

WHAT'S IN THIS FOLDER
- "dist" folder ......... the finished map, ready to put online
- "data" file ........... the list of all 127 companies

HOW TO PUT IT ONLINE
You'll need whoever looks after the website to do this — it's a small job:
1. Give them the "dist" folder.
2. Ask them to upload it to the website.
3. Ask them to show it on the weapons map page (the page already has the right
   spot for it — nothing else on the page needs to change).

That's it. The web address stays the same as before.

HOW TO UPDATE THE COMPANY LIST
Open the file "data/weapons.geojson" in any text editor. Each company has a
name, address, program, and source. Add or change entries there — no other
changes needed.
